
# 📘 Tarefa: Jogo da Forca

## 🎯 Objective

Construa o clássico jogo de adivinhar palavras usando strings, loops e entrada de dados do usuário em Python.

## 📝 Tasks

### 🛠️ Jogo Principal

#### Descrição
Implemente a lógica principal do jogo para selecionar uma palavra aleatória, receber palpites do usuário e exibir o progresso atual.

#### Requisitos
O programa completo deve:

- Selecionar palavras aleatoriamente de uma lista predefinida
- Aceitar palpites de letras e mostrar o progresso atual no formato `_ _ _`
- Rastrear e mostrar tentativas incorretas restantes
- Atualizar a palavra oculta à medida que letras corretas são adivinhadas
- Finalizar o jogo quando a palavra for adivinhada ou as tentativas acabarem
- Exibir mensagens de vitória ou derrota

### 🛠️ Validação de Entrada

#### Descrição
Valide os palpites do usuário e garanta que apenas letras válidas sejam aceitas, além de evitar repetições.

#### Requisitos
O programa completo deve:

- Aceitar apenas entradas de uma única letra
- Ignorar entradas inválidas ou repetidas
- Informar o usuário quando um palpite já foi feito
- Manter o jogo funcionando até o fim do ciclo de tentativas
